Get up & running

Run it locally in 30 seconds

① Install
pip install --user -r requirements.txt
② Run
python3 app.py
# → http://localhost:5000
③ Rebuild Tailwind (only if you edit input.css)
npm install
npm run build:css

The API

JSON endpoints

All routes return JSON. Admin-only routes need an X-Admin-Key header.
